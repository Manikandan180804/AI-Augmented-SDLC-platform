# AI-Augmented SDLC Platform

A comprehensive platform designed to augment the Software Development Lifecycle with AI-powered modules.

## Modules
1. **Requirements Analyzer**: Ambiguity detection and Gherkin story generation.
2. **AI Code Reviewer**: Security-first architectural reviews and diff suggestions.
3. **Smart Test Generator**: Automated test suite generation.
4. **Deployment Risk Scorer**: Predictive risk analysis for production releases.
5. **Traceability Engine**: End-to-end mapping from requirements to deployment.
6. **Multi-Agent Bug Fixer**: Collaborative AI agents for rapid MTTR.
7. **AI System Architect**: Automated Mermaid.js architecture diagram generation.

## Tech Stack
- **Backend**: FastAPI, OpenAI GPT-4o, LangChain, FAISS.
- **Frontend**: React 18, Tailwind CSS / Vanilla CSS, Glassmorphism UI.

## Getting Started
### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python main.py`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
